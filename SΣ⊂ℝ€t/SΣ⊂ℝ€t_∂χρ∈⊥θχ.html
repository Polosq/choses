TUTO - Prendre le contrôle d'un ordinateur windows sans mot de passe :

  1. Aller dans le menu pour éteindre l'ordinateur.
  2. Laisser appuyé sur la touche "Shift" ou "Maj" et cliquer tout en laissant appuyé sur "Redémarrer".
    Cela ouvrira le mode de dépannage de l'ordinateur.
  3. Une fois dans le mode de dépannage, cliquer sur "Dépanner" puis sur "Options avancées".
  4. Cliquer maintenant sur "Invite de commandes".
  5. Une fois dans l'invite de commandes, taper la commande suivante :
    ren sethc.exe sethold.exe qui va renommer le service des touches rémanantes en 'sethold.exe'.
    Vous pouvez nommer 'sethold.exe' comme vous le souhaitez, il faut seulement que le nom soit différent de l'ancien et que vous le reteniez facilement pour remettre l'ordinateur à la normale ensuite.
  6. Par sécuritée, si vous le souhaitez, vous pouvez faire une copie de cmd :
    copy cmd.exe le_nom_que_vous_voulez
  7. Ensuite, renommez le fichier 'cmd.exe' en 'sethc.exe' :
    ren cmd.exe sethc.exe.
    Ainsi, quand vouez appuirez 5 fois sur la touche 'Shift' ou 'Maj', l'invite de commande s'ouvrira en mode administrateur, que vous soyez connectés a un compte ou non.
  8. Pour finir, redémarrez l'ordinateur et appuyez 5 fois sur la touche 'Shift' ou 'Maj' pour ouvrir l'invite de commande.
    Pour que chaque commande s'exécute ou soit prise en compte, il faut parfois redémarrer l'ordinateur.
Pour aller plus loin :
Un compte adminstrateur existe sur chaque ordinateur par défaut :
Il est possible de le réactiver en tappant : net user Administrateur /active:yes puis net user Administrateur* qui vous permettra de mettre le mot de passe de votre choix pour ce compte.
Enfin, n'oubliez pas de redémarrer l'ordinateur afin que les changements soit appliqués

ATTENTION : Cette méthode peut être illégale et peut causer des problèmes si vous ne savez pas ce que vous faites. Utilisez-la à vos risques et périls.
