Comment modifier son curseur ?

  1. Choisir un cuseur sur le site favorisxp, Deviant Art, ou encore le sub reddit r/cursor. Vous pouvez aussi en créeer un avec l'application RealWorld Cursor Editor.
  2. Après avoir trouvé le curseur et l'avoir téléchargé, aller dans les paramètres de la souris de windows

    puis cliquez sur paramètres supplémentaires.
  3. Cliquez sur "Pointeurs", cliquez sur aucun dans les modèles, puis modifier chaque curseur un par un.
  4. N'oubliez pas d'appuyer sur "enregistrer sous..." et de donner un nom.
  5. Appuyez enfin sur appliquer.

Créer une extension web
  1. Créer un fichier manifest.json.
    Créer un fichier popup.html pour l'interface utilisateur.
    Créer un fichier popup.js.
    Créer un fichier background.js (si besoin uniquement !).

  3. Penser a créer les 3 icones aux différentes tailles (16*16 ; 48*48 ; 128*128), de vérifier que la version du manifeste est bonne, et que les permissions nécessaires sont bien écrites.
  4. Compléter le fichier background.js. Afin de le compléter : il est conseillé de chercher la syntaxe sur internet.
  5. Compléter le fichier popup.html : ce sera la page qui sera affichée quand l'utilisateur cliquera sur l'extension depuis son navigateur.
  6. Compléter enfin le fichier popup.js qui contiendra les codes principaux en lien avec la popup.
  7. Pour Firefox :
    • Compacter le fichier au format '.zip' puis rennomer afin de mettre au format '.xpi'.
    • Eberger l'extension sur un site et créer un lien pour pouvoir y accéder.
    Pour Chrome (et les autres navigateurs compatibles):
    • Afin que l'extension soit dissponible pour tous, il faut l'installer sur le Chrome Web Store. Pour cela, il faut que les équipes de Chromes valident l'extension et poseront peut être des questions.
    • Si vous n'avez pas envie de faire des démarches, vous pouvez aller sur le panneau des extensions, activer le mode développeur et donner le fichier contenant votre extension ou le fichier manifest.json en fonction du navigateur.
